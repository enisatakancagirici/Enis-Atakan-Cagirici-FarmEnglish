2. Information We Collect

Classic Mode

In Classic Mode, no personal data is collected. All progress data is stored locally on the user's device and is never transmitted to any server.

Voice Mode (SesYap)

In Voice Mode, audio captured from the microphone is sent to the Google Speech-to-Text API for pronunciation evaluation. This audio:

  • Is not sent to or stored on the developer's server
  • Is processed solely through Google's API infrastructure
  • Is subject to Google's Privacy Policy: policies.google.com/privacy

Battle Mode (Online Mode)

Battle Mode uses Firebase infrastructure for real-time online gameplay. Only the following data is processed:

  • Username: The nickname chosen by the user at registration

This is used solely for displaying scores on the leaderboard to other players. No real name, email, location, or other personal information is collected.

3. Purpose of Data Processing

  • Providing core application functionality
  • Saving user progress locally
  • Pronunciation evaluation (Voice Mode only)
  • Online leaderboard and matchmaking (Battle Mode only)

4. Data Storage and Security

  • Classic Mode data remains solely on the user's device
  • Voice Mode audio is not stored on the developer's server
  • Battle Mode stores only the username on Firebase
  • Deleting the application permanently removes all locally stored data

5. Third-Party Services

  • Google Speech-to-Text API – Pronunciation recognition (Voice Mode)
  • Firebase (Google LLC) – Online matchmaking and username storage (Battle Mode)

6. Data Sharing

FarmEnglish does not sell or share personal data for advertising or marketing purposes. Audio is processed technically only through Google's API; usernames are held on Firebase solely for gameplay functionality.
