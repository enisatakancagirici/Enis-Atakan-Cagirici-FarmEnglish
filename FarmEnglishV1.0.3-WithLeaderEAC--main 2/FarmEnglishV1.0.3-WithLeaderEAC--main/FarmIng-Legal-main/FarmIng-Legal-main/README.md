# FarmIng Legal Documents

This repository contains the legal documents for the **FarmIng** mobile application.

## 📄 Available Documents

- **[index.html](index.html)** - Main page with links to all legal documents
- **[kvkk.html](kvkk.html)** - KVKK Aydınlatma Metni (Turkish)
- **[privacy.html](privacy.html)** - Privacy Policy (English)

## 🌐 Hosting

These HTML files can be hosted on:
- **GitHub Pages** - Free hosting
- **Netlify** - Free hosting
- **Any web server**

### Setup with GitHub Pages

1. Create a new repository named `FarmIng-Legal`
2. Push these files to the `main` branch
3. Go to Settings → Pages
4. Select "Deploy from a branch"
5. Choose `main` branch and `/root` folder
6. Your site will be available at: `https://yourusername.github.io/FarmIng-Legal/`

## 🔗 Links for Mobile App

Update the ProfileScreen in the mobile app to use:

```
https://yourusername.github.io/FarmIng-Legal/privacy.html
https://yourusername.github.io/FarmIng-Legal/kvkk.html
```

Or use raw GitHub links:

```

https://raw.githubusercontent.com/yourusername/FarmIng-Legal/main/kvkk.html
```

## 📝 Update Instructions

To update any legal document:

1. Edit the `.html` files
2. Commit changes: `git add . && git commit -m "update: legal documents"`
3. Push: `git push origin main`
4. Changes will be live immediately (or within a few minutes)
