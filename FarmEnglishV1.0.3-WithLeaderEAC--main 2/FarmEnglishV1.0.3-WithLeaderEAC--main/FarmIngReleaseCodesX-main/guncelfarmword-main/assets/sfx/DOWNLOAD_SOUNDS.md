## İndirdikten Sonra

1. Tüm dosyaları `assets/sfx/` klasörüne koy
2. Dosya isimleri tam olmalı: correct.mp3, wrong.mp3, vs.
3. Expo'yu restart et: `npx expo start --clear`
4. Sesler otomatik yüklenecek!
